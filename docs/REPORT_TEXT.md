# Текст для отчёта

## Название

Веб-система управления сетью сайтов с SEO-мониторингом и планированием перелинковки (SiteNet Manager)

## Описание

Приложение для ведения группы сайтов в одной SEO-сети. Можно создавать сети и сайты, запускать проверку title/description/h1, смотреть доступность по HTTP-коду, строить перелинковку и выгружать sitemap и CSV. Есть модуль ключевых слов и блок по поведенческим метрикам (время на сайте, отказы, глубина просмотра).

Стек: React, Node.js, Express, Prisma, PostgreSQL.

## Use Case 1 — Создание сети

Вход → «Сети» → название и описание → «Создать». Сеть появляется в списке.

## Use Case 2 — Добавление сайта

«Сайты» → домен, название, ниша, теги → сохранить.

## Use Case 3 — SEO-анализ

Карточка сайта → «SEO-анализ» → сохраняются title, description, h1, оценка.

## Use Case 4 — Матрица ссылок

Сеть → матрица → источник, цель, анкор → связь в БД, экспорт CSV.

## Дополнительно

На главной — HTTP-статусы всех сайтов. В карточке сайта — семантическое ядро (частотность, конкуренция, сложность продвижения) и анализ поведенческих показателей с рекомендациями по доработке контента.

## Рефлексия

Спроектировал БД в Prisma, сделал REST API и интерфейс на React. Сложнее всего было парсинг SEO и отображение матрицы ссылок. Разобрался с JWT, деплоем на Vercel/Render и миграциями PostgreSQL.
